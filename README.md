</pre>
<h2>umijs</h2>
<pre>
umi，中文可发音为乌米，是一个可插拔的企业级 react 应用框架。umi 以路由为基础的，支持类 next.js 的约定式路由，以及各种进阶的路由功能，并以此进行功能扩展，比如支持路由级的按需加载。
帮助文档:<a href="https://umijs.org/zh/guide/" target="_blank">umijs</a>
</pre>
